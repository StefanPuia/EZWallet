# EZWallet

Install
-------
1. Download Node.js 
https://nodejs.org/en/

2. Download Git
https://git-scm.com/downloads

3. Clone repository
```bash
git clone https://github.com/StefanPuia/INSE-EZWallet.git
```

4. Install modules
```bash
cd INSE-EZWallet
npm install
```

Testing
-------
To run the server:
```bash
npm server
```

To stop the server (Ctrl+C or Command+C):
```bash
^C
```
